# Antidepresivos 2026 — Edición Profesional

**Antidepresivos 2026** es una Herramienta de Soporte Clínico Inteligente diseñada para profesionales de la salud mental. Combina una base de datos farmacológica rigurosa con utilidades interactivas para mejorar la toma de decisiones clínicas.

![Preview](https://antidepresivos.drcelada.com/assets/icons/icon-512.png)

## 🚀 Características Principales

- **📦 Monografías Detalladas**: Perfiles completos de farmacocinética, farmacodinámica, seguridad y dosis.
- **🔄 Calculadora de Switching**: Generador de planes de transición entre fármacos basado en guías Maudsley y CANMAT.
- **⚖️ Comparador Avanzado**: Visualización radial (Radar) de perfiles de efectos adversos y riesgos (QT, Sedación, Peso).
- **🎮 PharmaQuiz (Modo Guru)**: Plataforma de repaso gamificada para residentes y especialistas.
- **📄 Exportación a PDF**: Generación instantánea de informes clínicos y certificados de experto.
- **📱 PWA & Offline**: Instalable como aplicación nativa y funcional sin conexión a internet.

## 🛠️ Tecnologías

- **Frontend**: Vanilla JS, HTML5, CSS3 (Material Design 3).
- **Librerías**: Chart.js, Canvas-Confetti, html2pdf.js, ZingTouch.
- **DevOps**: Jest (Testing), GitHub Actions (CI/CD), Wrangler (Cloudflare Pages).

## 📦 Instalación y Desarrollo

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Ejecutar tests
npm test
```
